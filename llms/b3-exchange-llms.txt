# B3 (Brasil Bolsa Balcão)

> B3 S.A. - Brasil, Bolsa, Balcão is the Brazilian exchange and financial market infrastructure operator. Its B3 for Developers portal documents 114 B2B REST APIs (OAuth 2.0 client credentials and ROPC, Mutual TLS with certificate pinning) across investor-area, OTC (Balcão), listed-markets, Tesouro Direto, Banco B3, and insurance domains. Access is contract-gated for institutions - no self-serve keys. Real-time market data flows over the UMDF multicast feed (FIX/FAST and Binary SBE) via authorized distributors; end-of-day and reference data ships as UP2DATA files.

Generated by API Evangelist from the b3-exchange repository (apis.yml + harvested artifacts). Not published by B3.

## APIs

- [B3 for Developers portal](https://developers.b3.com.br/): catalog of 114 REST APIs across nine categories
- [B3 Investor Area APIs](https://developers.b3.com.br/apis/api-area-do-investidor): D-1 investor positions, transactions, corporate events (consent-based)
- [B3 Tesouro Direto APIs](https://developers.b3.com.br/apis/tesouro-direto): Treasury Direct bonds reference data, positions, and orders
- [B3 OTC (Balcão) APIs](https://developers.b3.com.br/apis/api-balcao): OTC registration, custody, and public-info consultation
- [B3 Listed Markets APIs](https://developers.b3.com.br/apis/api-listados): iMercado, CORE risk calculation, securities lending, reconciliation
- [Banco B3 APIs](https://developers.b3.com.br/apis/bancob3): custody reports, fund quota validation, Pix settlement
- [B3 Insurance APIs](https://developers.b3.com.br/apis/seguros): insurance registration (co-insurance, claims, documents)
- [B3 Authentication APIs](https://developers.b3.com.br/apis/autenticacao): OAuth 2.0 token issuance (client credentials + ROPC variants)
- [UMDF Market Data Feed](https://www.b3.com.br/en_us/market-data-and-indices/data-services/market-data/): real-time multicast FIX/FAST + Binary SBE (not HTTP)
- [UP2DATA](https://www.b3.com.br/en_us/market-data-and-indices/data-services/up2data/): end-of-day and reference data files
